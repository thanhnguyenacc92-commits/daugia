📖 Hướng dẫn cài đặt

1
Deploy Cloudflare Worker (proxy) npm install -g wrangler cd worker/ wrangler deploy

Lấy URL từ output, dán vào "Proxy API URL" ở trên

2
Đăng ký Yahoo Japan Developer

Vào developer.yahoo.co.jp → Đăng ký App ID miễn phí

3
Cài đặt EmailJS

Vào emailjs.com → Tạo account → Kết nối Gmail → Tạo template với các biến: {{title}}, {{price}}, {{url}}, {{items}}

4
Deploy PWA lên GitHub Pages git init git add . git commit -m "Initial commit" git push origin main

Vào Settings → Pages → chọn branch main
